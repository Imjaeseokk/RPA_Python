# CNN Fear And Greed Index Mailing

Python Selenium으로 [CNN Fear and Greed Index](https://edition.cnn.com/markets/fear-and-greed)를 Scrapping 후 Mailing 하는 Project 입니다.

해외 주식 투자에 참고하기 위한 보조 지표로서,

매일 해당 페이지에 접속하여 확인하는 시간을 줄이고자 매일 특정 시각 메일로 받아보고자 함





Local에서는 Scrapping + Mailing 잘 동작하나

매일 일정한 시각에 동작하도록 Github Actions에서 실행해보니
